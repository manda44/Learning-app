7 TYPES DE NOTIFICATIONS DISPONIBLES:

1. COURSE_UPDATE (Bleu)
   Objectif: Mises à jour de cours
   Destinataires: Étudiants
   Priorité: Basse

2. ENROLLMENT_CONFIRMATION (Vert)
   Objectif: Confirmation d'inscription
   Destinataires: Étudiants + Admin
   Priorité: Très Basse

3. QUIZ_REMINDER (Orange)
   Objectif: Rappel de quiz
   Destinataires: Étudiants + Admin
   Priorité: HAUTE

4. GRADE_RECEIVED (Violet)
   Objectif: Notes reçues
   Destinataires: Étudiants + Admin
   Priorité: Moyenne

5. PROJECT_FEEDBACK (Cyan)
   Objectif: Feedback de projet
   Destinataires: Étudiants + Admin
   Priorité: HAUTE

6. ADMIN_MESSAGE (Indigo)
   Objectif: Annonces admin
   Destinataires: TOUS
   Priorité: Haute

7. SYSTEM_ALERT (Rouge)
   Objectif: Alertes système
   Destinataires: Concernés
   Priorité: URGENTE
